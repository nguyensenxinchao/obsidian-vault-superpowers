# Summarize and Capture

Prompt to summarize a URL, article, or document → knowledge note.

---

## Prompt

```
Read the following content and create a knowledge note for my Obsidian vault:

URL / Source: [URL or document name]

Content:
---
[PASTE CONTENT]
---

Requirements:
1. Write a 1-paragraph Summary (in my own voice — not copied verbatim)
2. List the most important Key Points (3-7 points)
3. Record notable Quotes / Excerpts (if any)
4. Write My Takeaways — why is this relevant to me?
5. Suggest: type (knowledge/resource), domain, stage

Output: frontmatter + full note content following vault schema.
```

---

## When to use

- Reading a long article and want to save the essence
- Watching a video that has a transcript
- Researching a new topic
