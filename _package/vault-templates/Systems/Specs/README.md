# Systems/Specs

Where **spec documents** and **research plans** are saved, created by:
- `/brainstorm` → `YYYY-MM-DD-<topic>-spec.md`
- `/plan` → `YYYY-MM-DD-<topic>-plan.md`

---

## Rules

1. Files here are **thinking artifacts**, not knowledge notes
2. Do not move to `Knowledge/` — these are working documents, potentially incomplete
3. Review periodically (monthly): specs that have been fully executed → archive or delete
4. Specs that have produced Evergreen notes → can be deleted (knowledge has been distilled)
